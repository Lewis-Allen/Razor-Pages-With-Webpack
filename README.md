# Razor Pages With Webpack
Template repository for a .NET Razor Pages application with Webpack and Node.js.
## Features:
Template makes use of:
* .NET 6
* Node.js
* Webpack
* TypeScript
* SCSS
## Assets
Assets are generated and placed into the wwwroot/dist/ folder.
