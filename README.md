# Razor Pages With Webpack
Template repository for a .NET Razor Pages application with Webpack.

## Features:
Template makes use of:
* .NET 6
* Webpack 5
* TypeScript & SCSS Support
* Bootstrap 5

## Assets
Assets are generated and placed into the wwwroot/dist/ folder.
